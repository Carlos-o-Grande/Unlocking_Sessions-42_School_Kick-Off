# Unlocking Sessions — 42 School Kick-Off

Structured, hands-on sessions covering essential programming concepts and tools for students at 42 School.

## Session Overview

### [Debugging in C](Debugging/)
Learn the fundamentals of debugging C programs using GDB (GNU Debugger) and Visual Studio Code. Topics include:
- GDB command-line interface and essential commands
- Setting breakpoints and inspecting program state
- Debugging with Visual Studio Code integration
- Identifying and fixing common C programming errors

### [C Syntax & Fundamentals](Syntax/)
Master core C programming concepts and best practices. Covers:
- Binary representation and the byte model
- C type system and memory contracts
- Compiler flags and defensive programming
- Boolean logic, operator precedence, and scope
- Loops, clean code conventions, and declaration syntax
