# Unlock Session: C Syntax & Fundamentals

A comprehensive introduction to C programming fundamentals and best practices for 42 School students.

##  Session Contents

### **Interactive Presentation** &nbsp; [(Unlocking_session_Syntax.html)](Unlocking_session_Syntax.html)
Master essential C concepts through an interactive, hands-on presentation. Covers:
- **First Byte** — Binary representation and bit manipulation
- **Vocabulary** — Function anatomy and C terminology
- **Byte & Types** — Memory model and type system
- **Compiler Flags & Makefile** — Build safety and configuration
- **Boolean Logic & Precedence** — Truth tables and operator ordering
- **Scope & Lifetime** — Variable boundaries and initialization
- **Loops** — While loops and iteration patterns
- **Clean Code** — Naming conventions and style
- **Defensive Programming** — Guard clauses, const, and safe pointer iteration
- **Reading C** — The cdecl method for complex declarations
